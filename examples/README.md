# Overview
**poweretl** is library to support database, warehouse/lakehouse python projects. *poweretl-examples* shows how to use library. All examples has been implemented as tests.


# Examples

| Name               | Description |
| ------------------ | ----------- |
| [Commands Splitter](tests/utils/file/commands_splitter/) | Create Version-Command output based on the text files and parameters.
| [Files Merger](tests/utils/file/files_merger/) |  Merge many files and parameters into one output.
| [Databricks](tests/databricks) |  Read about prerequisites for run Databricks examples.
| [Databricks Model Manager](tests/databricks/managers/dbx_model_manager/) |  Provision simple model to databricks using json configuration files and yaml parameters.
